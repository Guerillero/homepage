ArcMap and .lock files

17/10/2016

Just as a heads up to all of my programer friends. IPv6 vs IPv4 will throw a curl: (35) Unknown SSL protocol error in connection to ${server} in response to a curl -v https://server.example.com on Windows Server 2012 R2. Sometimes, the fix is as simple as forcing IPv4 by using the -4 command. (curl -v -4 https://server.example.com).
